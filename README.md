# Angular 6 + Material

Angular 6 with material design template for Basic dashboard, Routing , HTTP call

### Concepts applied
* Basic Material Navbar
* HTTPClient for API calls
* Routing

### Prerequisites

Node JS > V.8
NPM > V.3



### Start Development Server


```
npm install
ng serve
```
